### Building with Cmake:
```bash
  mkdir -p build
  cd build
  cmake .. -L -DZ3_INCLUDE_DIR=/usr/include 
```

For API documentations, see the page [code-doc](https://rawgit.com/Bo-Yuan-Huang/Bo-Yuan-Huang.github.io/ilair/docs/ref-html/index.html).
